# Simulacro PSA · UMSA Derecho

Examen de práctica para la **Prueba de Suficiencia Académica** de la carrera de Derecho — Universidad Mayor de San Andrés (UMSA), La Paz, Bolivia.

## 📚 Contenido

100 preguntas de opción múltiple divididas en 4 categorías:

| Categoría | Descripción |
|-----------|-------------|
| **CPE** | Constitución Política del Estado Plurinacional |
| **DUDDHH** | Declaración Universal de Derechos Humanos |
| **Rousseau** | Discurso sobre el origen de la desigualdad |
| **Historia** | Historia de Bolivia |

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador, o
2. Despliega en GitHub Pages para acceso en línea.
